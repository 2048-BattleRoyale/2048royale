What is this?

2048Royale is an innovative battle royale game conceptualized and started at MVHACKS 2019, written in Vue.JS, Bootstrap, Anime.JS, and JQuery. The game is maintained by Aidan Smith and Neil Balch, who have developed the client-side and server side respectively. Noticed a bug? Report it here. If you wish to look at the source code for the project, you can find that here.
